Robot de détection d’obstacles en temps réel .

Utilisation de YOLOVn8 + Flask sur Raspberry Pi

Résumé technique

  • Technologies : Raspberry Pi 4, Python, YOLO, OpenCV, Flask
  • Domaine : Robotique embarquée, vision par ordinateur
  • Objectif : Détection d’obstacles en temps réel et affichage via une interface web

Présentation du projet

Ce projet consiste à développer un système embarqué capable de détecter en temps réel les obstacles à l’aide du modèle de vision par ordinateur YOLO. Le traitement est effectué sur un Raspberry Pi et les résultats sont diffusés via une interface web créée avec Flask. L’objectif est de démontrer l’intégration entre intelligence artificielle et robotique pour la navigation autonome.
